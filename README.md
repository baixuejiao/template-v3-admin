## 下载&安装

- 安装依赖

  ```bash
  npm install
  ```

- 运行本地开发环境
  
  ```bash
  npm run dev
  ```

- 本地运行测试环境

  ```sh
  npm run dev:test
  ```

- 本地运行正式环境

  ```sh
  npm run dev:prod
  ```

- 部署测试环境

  ```sh
  npm run staging
  ```

 - 部署正式环境

  ```sh
  npm run build
  ```
